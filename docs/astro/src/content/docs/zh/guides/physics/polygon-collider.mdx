`PolygonCollider` 定义最多 8 个顶点的凸多边形形状。适用于无法用盒形、圆形或胶囊形表示的自定义碰撞形状。

## 属性

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `vertices` | Vec2[] | 单位正方形 | 多边形顶点，以物理单位计（局部空间）。最多 8 个顶点。默认：`[{x:-0.5,y:-0.5}, {x:0.5,y:-0.5}, {x:0.5,y:0.5}, {x:-0.5,y:0.5}]` |
| `radius` | number | `0.0` | 多边形外围的蒙皮半径。大于 0 时创建圆角多边形 |
| `density` | number | `1.0` | 质量密度 |
| `friction` | number | `0.3` | 表面摩擦力 |
| `restitution` | number | `0.0` | 弹性 |
| `isSensor` | boolean | `false` | 触发区域，无物理响应 |
| `enabled` | boolean | `true` | 启用/禁用碰撞体 |
| `categoryBits` | number | `1` | 碰撞过滤类别位 |
| `maskBits` | number | `0xFFFF` | 碰撞过滤掩码位 |

<Aside type="caution">
  Box2D 限制每个多边形最多 **8 个顶点**。超过此数量的多边形将被拒绝。如需更复杂的形状，请将其分解为多个凸多边形或使用 [ChainCollider](./chain-collider/) 处理静态边界。
</Aside>

## 场景视图中的顶点编辑

在编辑器中选中带有 `PolygonCollider` 的实体时，场景视图中会显示顶点控制点：

- **拖拽控制点**移动顶点
- **双击边**添加新顶点（在 8 顶点限制以内时）
- **右键点击控制点**删除顶点

<Aside type="note">
  顶点必须构成**凸多边形**。编辑器会强制保持凸性——如果拖拽会产生凹形状，顶点位置会被约束。
</Aside>

## 用法

```typescript
commands.spawn()
    .insert(Transform)
    .insert(RigidBody, { bodyType: 2 })
    .insert(PolygonCollider, {
        vertices: [
            { x: -0.5, y: -0.5 },
            { x: 0.5, y: -0.5 },
            { x: 0.3, y: 0.5 },
            { x: -0.3, y: 0.5 },
        ],
    });
```

## 提示

- 保持多边形简单。顶点越少性能越好。
- `radius` 属性在多边形外围添加蒙皮，可减少堆叠场景中的碰撞抖动。
- 对于凹形状，将其拆分为多个凸 `PolygonCollider` 实体，挂载到同一父实体下。
