<Aside type="tip">
v0.6.1 专注于播放模式可靠性提升，新增项目模板浏览器，修复多个渲染和编辑器问题。
</Aside>

## 模板浏览器

- 新建项目对话框新增模板浏览器，支持从模板创建项目
- 内置 **Space Shooter** 示例项目作为首个模板
- 模板以 zip 包形式打包，创建项目时自动解压

## 播放模式改进

- 用户脚本现在在共享播放模式中执行 — 自定义组件和系统直接在编辑器中运行
- 双向 Inspector 同步：编辑运行时实体属性并实时查看值更新
- 播放模式下 Inspector 正确显示资产字段（纹理、材质），不再显示原始数字句柄
- 暂停和速度状态在编辑器控件和 SDK `App` 实例之间正确同步
- 暂停状态下 Step Frame 按钮现在可以正常工作

## Bug 修复

### 渲染

- 修复 UIRect pivot 偏移导致的精灵渲染错位 — 非中心 pivot 不再导致视觉偏移
- 修复编辑器 UI 元素的 Gizmo 对齐 — Gizmo 现在考虑 pivot 到中心的偏移，与渲染位置一致

### 编辑器

- 修复嵌套预制体导航：打开嵌套预制体前先保存当前预制体
- 修复 Game View 面板的键盘输入处理
- 新项目自动创建默认 `asset-groups.json`

### SDK

- 修复使用 `assetServer.baseUrl` 作为场景资产预加载的回退路径
- 为 `AssetServer` 添加资产引用解析器，支持基于 UUID 的预制体引用

## 内部改动

- 预制体文件现在在磁盘上以 UUID 存储资产引用，保持一致性
- 引擎和 SDK 版本号作为构建时定义暴露
- 移除 `GameViewBridge`，改为直接集成播放模式服务
